<h3 align="center">The chat app made by otakus, for otakus 🎌</h3>

**Goldiss Chat** is a real‑time messaging application built for fans of Japanese pop culture.  
Share your passion for anime, manga, games, and everything weeb – in a space that feels like home.

---

## ✨ Key Features

- 🎎 **Otaku‑themed experience** – Custom stickers, emoji packs, and chat backgrounds inspired by Japanese pop culture.
- ⚡ **Real‑time messaging** – Instant one‑on‑one and group conversations powered by Supabase.
- 🔔 **Smart notifications** – Firebase Cloud Messaging keeps you updated even when the app is in background.
- 🧩 **Rich profiles** – Show off your favorite anime, manga list, and a custom status.
- 🗨️ **Rooms & communities** – Create or join public rooms by genre (shonen, shojo, isekai…).
- 🧠 **State management with Riverpod** – Clean, testable, and scalable architecture.
- 🔒 **Secure authentication** – Supabase Auth with email/password and social login (Google, Discord).
- 🌙 **Light & dark mode** – Because we know you code at 3am watching anime.

---

## 🚀 How It Works

1. **Sign up** – Create your otaku profile.
2. **Find friends** – Search by favorite anime or language.
3. **Chat freely** – Send messages, stickers, images, and voice notes.
4. **Join communities** – Discuss the latest episode or share fan art.
5. **Stay notified** – Never miss a message thanks to Firebase push notifications.

---

## 🧰 Tech Stack

| Layer | Technology |
| --- | --- |
| Frontend | Flutter 3.x, Dart |
| State Management | Riverpod |
| Backend / Database | Supabase (Postgres, Realtime, Auth) |
| Notifications | Firebase Cloud Messaging |
| Storage | Supabase Storage (for media files) |
| Tools | VS Code, Android Studio, Git |

## ▶️ Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/joihomisael-dev/goldiss-chat.git
cd goldiss-chat

# Install dependencies
flutter pub get

# Run on a connected device or emulator
flutter run
